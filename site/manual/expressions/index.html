    2. latlon2h3¶

    Convert (lat, long) to H3 ID.

    Syntax

  2. latlon2h3(lat, long, resolution [0..15]) or latlon2h3($y, $x, resolution) in WGS84 CRS

    3. Example usage

  3. latlon2h3(10.775275567242561, 106.70679737574993, 13)→ '8d65b56628e46bf'


    4. latlon2s2¶

    Convert (lat, long) to S2 Token.

    Syntax

  4. latlon2s2(lat, long, resolution [0-->30]) or latlon2s2($y, $x, resolution) in WGS84 CRS

    5. Example usage

  5. latlon2s2(10.775275567242561, 106.70679737574993, 21)→ '31752f45cc94'
